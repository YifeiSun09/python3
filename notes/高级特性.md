# 高级特性  
1. 切片  
从列表或者元组中取出部分元素.
负数表示尾部位置(-1表示最后一个,-2表示倒数第二个,以此类推.)  
```python
L = [1,2,3,4,5,6,7,8]

print(L[1:]) # 2,3,4,5,6,7,8

print(L[:-1]) # 1,2,3,4,5,6,7

print(L[2,3]) # 3

# 取出前两个数组成的列表
print(L[:2]) # 1,2
```  
利用切片实现trim函数:  
```python
def trim(s):
    if len(s) > 0:
        if s[0] == ' ':
          return trim(s[1:])
        elif s[-1] == ' ':
          return trim(s[:-1])
    return s
```
